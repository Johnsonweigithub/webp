畢業專題-CGU動態查詢系統

簡介

Dcard是大學生常使用的匿名社群網站，大學生會在上面發布有關於學校、課業、日常等等的相關文章。因此學校輔導組希望我們可以透過爬取Dcard上有關長庚大學的文章來及時取得同學們對學校的想法或者是找出情緒不穩定的學生。

畢業專題的主要工作便是透過APP來統整及圖表化呈現從Dcard上的文章內容及對應的情緒分數(先前由碩班學長完成情緒分數的計算)。個人在小組裡面的分工主要為UI、UX的設計，在構思上主要規劃了主頁(包含重要通知、最新資料)、各主題式資訊的獨立頁面及提供校方更加管理途徑的管理者頁面等。另外，也針對長庚大學的主題色-橘色來設計整個APP的配色。

入口頁面

        登入頁面分為一般使用者登入及管理者入，管理者可以對使用者資料進行新增及查詢。此頁面背景顏色使用的是長庚大學的主題色，並且使用毛玻璃的效果讓整體頁面看起來較有立體感且不單調。

管理者之首頁

        管理者登入後便會看到現有之所有帳號資料。

管理者側邊選單

        管理者可以透過側邊選單來切換到新增帳號頁面或更改自己密碼的頁面。

新增帳號頁面

        管理者可以在此輸入要新增之帳號的所有相關資料，並且指定該帳號是否可以擁有管理者之權限。

新增確認提示訊框

        讓管理者再次確認所有資料並無輸入錯誤。

使用者主頁

         使用者登入後的入口頁面。最上面為四個快捷鍵，可以讓使用者快速到達主頁中該資料區塊。快捷鍵下方專門做為及時推播的區塊，讓學校可以及時推廣重要訊息。日、周、月三個按鈕讓使用者可以自行控制主頁中顯示的概覽資料之時間區段(以當天往回推)。

使用者主頁-文章概覽

        顯示最新的五筆文章之標題、部分內容、左下的情緒分數及右下的發布時間。按下右上方的按鈕可以前往文章頁面。

使用者主頁-情緒概覽

        顯示使用者所選時間區段內所有文章之情緒分數統計狀況。綠色為正向、黃色為一般、粉紅色為負向。情緒趨勢利用柱狀圖顯示從當下時間往回推算一年當中每個月的平均情緒分數。

使用者側邊選單

        讓使用者可以快速跳轉到各重點頁面，以及更改密碼和登出。

文章總覽

        從新到舊展示文章的重要訊息，包含標題、正負向分數、日期等。使用者可以點擊某一篇文章，跳轉到該文章頁面以閱讀全文。

個別文章頁面

        使用者點進來就可以看到包含文章完整內容等全部資訊。左上角為標題，右上角按鈕讓使用者可以跳轉到Dcard該篇文章位址。標題下方有的彩色字樣第一排代表本篇文章之情緒分數，第二排代表文章中是否有比對到特殊情緒關鍵字，如:憂鬱、自殺等。

情緒總量頁面

        情緒總量頁面可以看到該段時間內(從主頁選擇的日、月、年)的三種情緒數量統計，並且透過點擊圓餅圖中不同情緒顏色可以切換下方文章的展示內容，如:點擊綠色區塊，下方便會只顯示正向情緒分數文章。
